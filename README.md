*Домашнее задание №16*

**Тема** ***"PAM"***

**Задача**
Запретить всем пользователям, кроме группы admin логин в выходные (суббота и воскресенье), без учета праздников

**Результат выполнения**
1. Создан Vagrant-file для стенда
```
vagrant status
Current machine states:

pam                       running (virtualbox)
```

2. Создана ansible роль, реализующая:
    - создание пользователей и добавление его в группу (Основываясь на карточке пользователя в defaults)
    - создание группы
    - добавление пользователей в группу
    - настройка PAM

3. В результате выполнение, в выходные к стенду могут подключаться к стенду только пользователи, состоящие в группе admin:
     
```
vagrant@pam:~$ date
Sat Sep 23 10:31:17 UTC 2023
vagrant@pam:~$ 
```
